Terminology

Service Catalog

An OpenShift feature that allows you choose the Mobile Service and Mobile Client that you want to implement. It is an implementation of the Open Service Broker API (OSB API) for Kubernetes that allows users to connect any of their applications deployed in OpenShift to a wide variety of service brokers.

Mobile Service

A mobile service is any feature of AeroGear Mobile Services that you want to implement, including:

  • Sync

  • Push

  • Auth

  • Metrics

  • API Management

  • CI/CD for Mobile Apps

Mobile Client

A mobile client is the representation on OpenShift of the Mobile App that you want to implement, including:

  • Android

  • iOS

  • Cordova
